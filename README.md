# Ecommerce

Este projeto consiste em um sistema de e-commerce desenvolvido em Python como parte da disciplina de Banco de Dados. Atualmente, ele também está sendo utilizado na disciplina de Testes e Qualidade de Software da Universidade Católica de Pernambuco.

## Sobre o Projeto

O sistema contempla funcionalidades essenciais de um e-commerce, como:

- Cadastro de produtos  
- Cadastro de clientes  
- Controle de permissões  
- Execução de vendas  
- Integração com banco de dados MySQL  

Durante o desenvolvimento, também foram implementados testes automatizados com o objetivo de validar o funcionamento das funcionalidades e analisar a qualidade do código.

## Resultados dos Testes

Se quiser visualizar o resultado do primeiro teste, ele está disponível em:

## Relatório de Cobertura

Para gerar o relatório de cobertura de testes, execute:
pytest --cov=ecommerce --cov-report=html test_ecommerce.py

Depois, para abrir o relatório no navegador:
firefox htmlcov/index.html


*(Execute após clonar o repositório)*
